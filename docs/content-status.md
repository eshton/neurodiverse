## Diagnosis: how the ~60-100 estimate was reached

Two separate institution types exist in Hungary, worth keeping distinct if this gets built out further:

1. **Medical diagnosis providers** (what this category models) — auti.hu's own directory lists ~20 autism-specific providers across 14 cities; ADHD Terápia's list shows ~8-10 state-supported ADHD sites. Neither list is exhaustive of private practices, which don't sit on any single registry — every search round this session surfaced several more (Panoráma Poliklinika, Soulwell, MENTA Szeged, Rovienne, CMed, PszichoFészek, Zöld Alma, Meteo Klinika, adhdautizmus.hu — the last of which was skipped for lack of a street address, only district-level location given). 60-100 is a rough synthesis of "state layer is ~20-30 and largely captured" + "private layer is a long uncounted tail, plausibly another 40-70."
2. **Tanulási Képességet Vizsgáló Szakértői Bizottságok** — official state committees, legally mandated one per county (some bigger counties run two) → ~20-25 nationally, fixed and well-documented. This is a *different* institution type (school-accommodation/SNI classification, not clinical diagnosis) and is **not** represented in the current 28 entries at all. Worth adding as its own thing if the goal becomes genuinely exhaustive, since it's a clean, finite, addable list.

## Research: assessment + how the category is scoped (2026-07-05)

New `research` category. The open question raised was: *ADHD/autism research is mostly in English — is there enough Hungarian content?* Assessment: **yes, if the net isn't Hungarian-only.** Three tiers exist — (1) genuinely Hungarian-language peer-reviewed work (Psychiatria Hungarica, Magyar Pszichológiai Szemle / `real.mtak.hu`, Gyógypedagógiai Szemle / `ojs.elte.hu`, EPA/OSZK), (2) Hungarian-authored / Hungarian-population studies published in English (Simon–Bitter Semmelweis group), and (3) landmark international English papers. 

**Decision (from the user): collect the best, most *relevant* research — Hungarian AND hugely-relevant international — each with a Hungarian-generated summary.** So it's a curated quality/relevance list, not "all autism research" and not "Hungarian-only." Publication types in scope: **peer-reviewed papers/reviews + clinical trials** (theses and clinical guidelines were considered but left out for now).

**Sourcing/verification techniques (see curation-methodology.md):**
- **Crossref API** (`https://api.crossref.org/works/<doi>`) for exact title/authors/year/venue of a DOI — clean, scriptable, no scraping.
- **ClinicalTrials.gov API v2** (`/api/v2/studies?query.cond=autism&query.locn=Hungary`) to find trials with *real* Hungarian sites, then `/api/v2/studies/<NCT>` for sponsor/site/summary. This is how the two trials were confirmed HU (don't trust a web-search claim of "HU site" — verify against the registry's location list).
- Open HU repositories give stable, linkable, free full text: `real.mtak.hu` (MTA), `ojs.elte.hu` (ELTE journals), `edit.elte.hu` (theses, if that type is ever added), `epa.oszk.hu`.

**Honesty rule extended:** summaries state the paper's *scope and main takeaway* in Hungarian; specific statistics are only included when verified from the abstract/paper. Every entry's body notes the source language and that the Hungarian summary is the editor's wording. Pharma-sponsored trials are labelled as such ("tájékoztatásul, nem gyógyszerajánlásként").

**Unexplored leads:** Stefanik et al. ADOS/ADI-R Hungarian adaptation (peer-reviewed version); more Psychiatria Hungarica / Orvosi Hetilap articles; Balázs Judit's comorbidity/suicidality studies; ENIGMA-ADHD/ASD imaging papers with HU co-authors; the CDC/among-international autism prevalence reviews. Category is a strong starter (8) but nowhere near exhausted.
