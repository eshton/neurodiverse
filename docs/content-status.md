## Diagnosis: how the ~60-100 estimate was reached

Two separate institution types exist in Hungary, worth keeping distinct if this gets built out further:

1. **Medical diagnosis providers** (what this category models) — auti.hu's own directory lists ~20 autism-specific providers across 14 cities; ADHD Terápia's list shows ~8-10 state-supported ADHD sites. Neither list is exhaustive of private practices, which don't sit on any single registry — every search round this session surfaced several more (Panoráma Poliklinika, Soulwell, MENTA Szeged, Rovienne, CMed, PszichoFészek, Zöld Alma, Meteo Klinika, adhdautizmus.hu — the last of which was skipped for lack of a street address, only district-level location given). 60-100 is a rough synthesis of "state layer is ~20-30 and largely captured" + "private layer is a long uncounted tail, plausibly another 40-70."
2. **Tanulási Képességet Vizsgáló Szakértői Bizottságok** — official state committees, legally mandated one per county (some bigger counties run two) → ~20-25 nationally, fixed and well-documented. This is a *different* institution type (school-accommodation/SNI classification, not clinical diagnosis) and is **not** represented in the current 28 entries at all. Worth adding as its own thing if the goal becomes genuinely exhaustive, since it's a clean, finite, addable list.

## Schools: how this round went

Search surfaced dedicated autism-specific schools (EGYMI-type state institutions, foundation schools) far more easily and reliably than "mainstream school that happens to integrate SNI kids" — the latter don't publicize it the way dedicated institutions do, so they're much harder to find and verify. That's why the 11 entries skew heavily `special-needs-school`. Worth deliberately hunting more `mainstream-integration` examples if this category gets a second pass — the one clean hit this round (Humánus) came from a Kölöknet roundup article explicitly using the word "integráltan," which is the search angle to repeat.

Ruled out / caught this round:
- **Kincsesház Általános Iskola** (Laborc utca 2, Óbuda) — initially a strong candidate (integrated autistic children, real address, real phone), but a bpn.hu news article confirms **the school closed** due to financial trouble. Caught before publishing — a reminder that "listed in a 2020s directory" isn't the same as "currently operating," always check for a closure story before trusting an aggregator page.
- **Laborc Általános Iskola** (Vihar utca 31, different institution than Kincsesház despite similar branding/area) — currently operating, run by Megérted Alapítvány, mentions "képességfejlesztés, terápiás foglalkozások" but no explicit autism/ADHD program could be confirmed. Skipped for now; worth another look if returning to this category with more time to verify.
- **Töbörzsöki / Szent István Tagiskola** (Sárbogárd) — real ADHD-integration story (feol.hu article, brought in the Hungarian ADHD Foundation president for staff training), but no street address surfaced anywhere, including its own site (blocked, 403). Skipped per scope discipline, same as the diagnosis category's `adhdautizmus.hu` precedent.
- Humánus's address needed correction mid-research: older directories (Kölöknet, docplayer) list Megyeri út 20/B, but the KIR (official state school registry) and a direct OSM/Nominatim name-match both confirm the school relocated to Kanizsai utca 6, XI. kerület. Trusted the registry + geocoded name-match over the older directory listings.
- Two geocoding traps hit this round, both resolved by adding more context to the query rather than trusting the first hit: "Liget utca 10" alone matched a same-named street in a different town (Jászalsószentgyörgy) instead of Szolnok — fixed by re-querying with the county name attached. "Rátz László utca" (Vadaskert Iskola) and "Thaly Kálmán utca" (Ferencvárosi EGYMI) both resolved to street-level only, landing near but not exactly on the right building (Nominatim matched an unrelated nearby POI by number) — accepted as city-scale-precise per the existing geocoding fallback rule, since neither address resolved to an exact building.
